## Requirements
- The modules package requires:
    - PHP 8.2 or higher
    - MySql 8.0 or higher

## Install
### Create project with composer
```
composer create-project --prefer-dist juzaweb/cms blog
```
### Install

Config database in your `.env` file, and run:

```
php artisan juzaweb:install
```
